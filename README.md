## 📖 Overview
This repository documents my continuous journey of mastering Data Structures and Algorithms (DSA) through [LeetCode](https://leetcode.com/u/Justinvcj/). 

It contains solutions to various algorithmic problems, categorized by difficulty and topic. Each solution aims to be highly optimized for both Time Complexity (Big O) and Space Complexity.

## ✨ Highlights
- **📊 Tracking Progress:** A structured archive of problems solved across Easy, Medium, and Hard difficulties.
- **🏗️ Core Patterns:** Implementations of fundamental DSA patterns including Sliding Window, Two Pointers, Fast & Slow Pointers, Merge Intervals, Cyclic Sort, and Tree BFS/DFS.
- **💡 Optimal Solutions:** Focuses on shifting from brute-force approaches to highly optimal mathematical or data-structure driven solutions.

## 🚀 Repository Structure
Solutions are generally organized by their problem number or core pattern topic. 

### Why this repo?
- To maintain a personal log of algorithmic growth.
- To act as a quick-reference guide for interview preparation.
- To showcase clean, readable, and efficient problem-solving skills in multiple languages.
